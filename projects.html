FPGA/Android Mobile - Scavenger Hunt Game
01/2016-04/2016
Design a fun application that includes the use of Nios II CPU (on the FPGA development board), LED touchscreen, bluetooth, GPS, and SD card.
The game design is very similar to the Amazing Race: you must solve a puzzle to move on to the next one. The FPGA act as geographic checkpoints that stores 'riddles'. The user's location is mapped on the Android app, and when user approaches, a connection is made via Bluetooth and user can then see and interact with the riddle on the touchscreen. There is also a timer feature and a leaderboard feature to further enhance the user experience.

My Roles:
• Wrote firmware to initiate and retrieve/send data for hardware components, then unit-tested hardware components for functionality.
• Developed C program to implement keyboard on LED Touchscreen
• Designed Intuitive UI on Android App.
• Mapped User locations on screen using Google API
